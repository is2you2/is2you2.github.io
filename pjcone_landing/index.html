About

Project Cone은 네트워크 연결이 불안정한 환경에서도 작동하는 오프라인 우선 협업 플랫폼입니다.
이메일 인증 없이 참여 가능한 익명 기반 구조를 통해, 자유로운 아이디어 교류와 실험적인 프로젝트 운영이 가능합니다.

Core Features

🧭 채널 기반 커뮤니케이션

팀별, 프로젝트별 대화를 채널로 분리하여 관리할 수 있으며, 개인 노트 공간도 제공합니다.

🎯 퀘스트 기반 작업 관리

작업을 퀘스트로 등록하고, 진행 상황을 시각적으로 추적할 수 있습니다.

🗂 파일 뷰어

문서, 이미지, 오디오, 3D 모델 등 다양한 형식을 직접 열람하고 그림판으로 편집하여 협업 효율을 극대화합니다.

⚙️ 오프라인 자동 동기화

네트워크 연결이 끊겨도 작업이 유지되며, 연결 시 자동으로 서버와 동기화됩니다.

🔐 자체 서버 운영

모든 데이터는 개인 또는 팀 서버에 저장되어, 완전한 자율성과 프라이버시를 제공합니다.

Get Started

별도의 설치 없이 브라우저에서 바로 실행할 수 있습니다.
PWA로 홈 화면에 추가하면 앱처럼 사용할 수 있으며, 서버 연결 시 팀 단위 협업이 가능합니다.
도커 기반 서버로 누구나 쉽게 컴퓨터에 서버를 설치하실 수 있습니다.
