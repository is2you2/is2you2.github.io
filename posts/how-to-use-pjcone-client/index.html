Project: Cone 사용 설명서 (v0.93.41)

Posted Dec 17, 2025
By 최성수 5 min read

🚀 Project: Cone (PJCone) 사용 설명서 v0.93.41

소규모 팀을 위한 익명 기반 Self-hosted 협업 및 미디어 공유 플랫폼 PJCone은 직접 서버를 구축하여 데이터 주권을 확보하고, 강력한 미디어 뷰어를 통해 팀원과 안전하게 소통하는 도구입니다.

1. 시작하기 (PWA 설치)

PJCone은 웹 브라우저 기반 앱으로, 별도의 설치 없이 사용하거나 PWA 기술을 통해 앱처럼 설치하여 사용할 수 있습니다.

  • 접속 주소: https://is2you2.github.io/pjcone_pwa/
  • 앱 설치 방법:
    • PC (Chrome/Edge): 주소창 우측 [설치] 아이콘 클릭
    • 모바일 (Android/iOS): 브라우저 메뉴의 [홈 화면에 추가] 기능을 이용해 설치 시 전체 화면으로 사용 가능합니다.

2. 핵심 철학 및 특징

  1. 완벽한 데이터 자율성 (Self-hosted): 사용자가 직접 개설한 서버에 데이터가 저장되어 보안과 프라이버시를 보장합니다.
  2. 익명성 및 유연성: 회원가입 없이 비회원 임시 초대가 가능하며, 익명 기반의 자유로운 소통을 지향합니다.
  3. 독립적 네트워크: 선택적으로 외부망과 차단된 폐쇄망 운영이 가능하여 팀만의 독립적인 공간을 구성합니다.

3. 주요 기능 가이드

📁 고성능 파일 뷰어 (Three.js 기반)

이미지, 영상은 물론 개발/디자인 팀에 최적화된 포맷을 강력하게 지원합니다.

  • 3D 모델링 지원: Three.js를 이용해 GLB, VRM 등 오픈 모델링 파일을 브라우저에서 즉시 열람할 수 있습니다.
  • 코드 파일: .java, .html, .md 등 구문 강조(Syntax Highlighting) 지원
    • .md 파일: 마크다운 프리뷰 지원
    • .html 파일: 소스 코드 결과물 즉시 확인 가능
  • 미디어 편집: 열람 중인 미디어를 간이 그림판으로 연결하여 즉시 수정하거나 실시간으로 공유할 수 있습니다.

🤝 프로젝트 관리 및 협업

  • 이슈 트래킹: 대시보드와 칸반 보드를 통해 작업의 진행 흐름(To Do, In Progress, Done)을 관리합니다.
  • QR 코드 공유: 앱 내 대부분의 정보는 QR 코드로 생성됩니다. 클릭하여 주소를 복사하거나 스캔하여 빠른 참여를 유도하세요.
  • 조커 탭: 작업 보조 및 비회원/불특정 다수와의 소통을 위한 특화 공간입니다.

4. 고급 조작법 (UX 가이드)

⌨️ PC 사용자 (단축키 및 마우스)

  • 단축키 힌트: 각 버튼 근처에 단축키가 표시됩니다. (설정에서 토글 가능)
  • 마우스 우클릭: 상세 메뉴/추가 동작 (모바일의 ‘길게 누르기’ 대응)
  • 휠 클릭: 특수 기능 (모바일의 ‘세 손가락 탭’ 대응)

📱 모바일 사용자 (제스처)

  • 길게 누르기 (Long Press): 상세 메뉴 호출 및 카드 속성 변경
  • 스와이프 (Swipe): 리스트 항목을 밀어서 삭제하거나 빠른 상태 전환

5. FAQ 및 주의사항

  • ⚠️ 서버 데이터 주의: 본 앱은 사용자가 호스팅하는 서버에 데이터를 저장합니다. 서버 관리 및 데이터 백업은 사용자가 직접 관리해야 합니다.
  • Q: 고도엔진 게임을 테스트할 수 있나요?
    • A: 네, HTML5 호환 .pck 파일을 뷰어에서 직접 열어 설치 과정 없이 즉시 테스트할 수 있습니다.
  • Q: 오프라인 모드를 지원하나요?
    • A: PWA 특성상 이전에 로드된 기본 데이터와 UI는 네트워크 연결 없이도 확인이 가능합니다.
