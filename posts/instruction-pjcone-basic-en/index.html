Project: Cone 공통 기능 설명서 [v0.85.2]

Posted Mar 8, 2025
By 그림또따 7 min read

앱 사용하기

이 프로그램은 웹 브라우저 기반 앱으로 이 페이지에 접속하여 사용할 수 있습니다.

개요

Project: Cone은 사용자가 직접 서버를 운영하는 내부망 메신저입니다.

소규모 게임 개발자 팀을 위한 소통 도구로 시작하였기에 보편적인 메신저와 비교했을 때 다소 생소한 기능들이 있습니다.

대표적인 기능 및 특징

  • 완벽한 익명성 추구
  • 사용자가 직접 서버 구성
  • 선택적 폐쇄망 운영 구조
  • 협업 및 미디어 뷰어 공유 도구
  • 비회원 임시 초대

공통 기능

이 페이지에서는 사용방식과 무관하게 앱을 사용할 때 알아두면 좋은 기능 및 정보에 대해서 다룹니다.

PC에서의 사용자 경험

앱의 모양새는 모바일에서 사용하기 적합하게 구성되어있습니다.

PC에서 사용할 때의 사용자 경험을 개선하기 위해 앱 내에서 할 수 있는 대부분의 행동은 단축키를 이용할 수 있도록 구성되어있습니다.

단축키 정보는 보편적으로 버튼 근처에 표시가 되며 “설정 > 단축키 힌트 표시” 기능을 토글하여 단축키 표시가 보여지지 않게 할 수 있습니다.

모바일에서 탭 행동은 PC에서 마우스 좌클릭, 길게 탭은 우클릭, 3손가락 탭은 마우스 가운데 클릭과 동일하게 동작합니다.

QR코드 공유

앱 내 대부분의 정보는 QR코드를 제공하고 있습니다. QR코드를 직접 스캔하거나, QR코드를 클릭하여 복사된 해당 주소를 공유하는 방식으로 빠른 참여를 유도하고 있습니다.

파일 뷰어

집중 모드를 사용하여 나머지 UI를 전부 가지로 미디어 감상에 집중할 수 있도록 도와주는 기능이 있습니다.

이 앱에서는 보편적으로 열람가능한 이미지, 음성, 영상 파일을 열 수 있으며 추가로 다음 종류의 파일들을 열 수 있습니다.

코드 파일

코드파일(예를 들면 *.java와 같은 것)을 열람한 경우 일부 확장자에 대해 구문 강조가 이루어져 코드를 읽을 때 도움이 될 수 있습니다.

*.html 과 *.md 파일에 대해 재생버튼을 지원합니다.

  • 재생버튼을 누르면 *.md 파일은 마크다운 프리뷰 보여줍니다.
  • *.html 파일은 해당 파일 결과물을 볼 수 있습니다.

Blender 파일 (실험적)

3D 모델링 툴 파일인 *.blend 파일을 파일 뷰어에서 열 수 있습니다. (<= 4.2.6)

4.3 버전부터는 파일 구조가 변경되어 지원하지 않고 있습니다. 최신 버전을 지원하지 못하기 때문에 이 기능이 완성된 것도 아니지만, 추가로 개발하기도 애매한 위치에 있습니다. 될 수 있으면 고도엔진으로 다시 출력하는 것을 권장합니다.

*.blend 파일이 앱에서 온전히 열릴려면 리소스를 *.blend 파일에 내장하고, rotation 값이 (0,0,0) 으로 구성되어야 합니다. 또한 triangulate 가 되어있어야 UV 맵핑이 깨지지 않고 표현됩니다.

smooth 처리가 이루어지지 않고 광원 표현이 제한됩니다.

고도엔진 패키지 파일 (실험적)

HTML5 저사양 버전에서 호환되는 결과물이라면 *.pck 파일을 파일 뷰어에서 직접 열람할 수 있습니다.

앱 설치파일을 공유하고, 설치하는 과정을 건너뛰고 파일 뷰어에서 직접 개발중인 게임을 테스트할 수 있습니다.

간이 그림판 연계

대부분의 경우, 열람된 미디어를 즉시 그림판으로 편집할 수 있는 기능을 제공합니다.

만약, 연결된 서버가 있다면 원격 그림판을 이용하여 다른 사람과 함께 그림을 그릴 수도 있습니다.

아케이드 탭

이 탭은 작업 보조용 탭으로 사용하거나 회원이 아닌 불특정 다수와 소통할 수 있는 기능들이 준비되어있습니다.
