An anonymous, self-hosted collaboration and media sharing platform for small teams. PJCone empowers teams to secure data sovereignty by hosting their own servers while providing powerful media viewers for seamless communication.

1. Getting Started (PWA Installation)

PJCone is a web-based app. You can use it directly in your browser or install it as an app using PWA technology.

  • Access URL: https://is2you2.github.io/pjcone_pwa/
  • How to Install:
    • PC (Chrome/Edge): Click the [Install] icon on the right side of the address bar.
    • Mobile (Android/iOS): Use the browser menu‚Äôs [Add to Home Screen] feature to use it in full-screen mode.

2. Core Philosophy & Features

  1. Total Data Autonomy (Self-hosted): Data is stored on a server you set up, ensuring maximum security and privacy.
  2. Anonymity & Flexibility: Supports guest invitations without registration, promoting free and anonymous communication.
  3. Independent Network: Can be operated as an optional closed network, creating an isolated space exclusive to your team.

3. Key Features Guide

üìÅ High-Performance Media Viewer (Three.js Powered)

Supports optimized formats for development and design teams beyond standard images and videos.

  • 3D Model Support: View GLB and VRM open modeling files directly in the browser via Three.js.
  • Code File Support: Syntax highlighting for .java, .html, .md, and more.
    • .md files: Markdown preview support.
    • .html files: Instant rendering of the code output.
  • Media Editing: Link any open media to the built-in Canvas for instant editing or real-time sharing with the team.

ü§ù Project Management & Collaboration

  • Issue Tracking: Manage task workflows (To Do, In Progress, Done) through dashboards and Kanban boards.
  • QR Code Sharing: Most information within the app can be converted into a QR code. Click to copy the URL or scan to invite others quickly.
  • Joker Tab: A specialized space for task assistance and communication with non-members or the general public.
