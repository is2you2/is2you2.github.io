Project: Cone 앱 사용 방법 (2)

Posted Feb 14, 2024
By 최성수 3 min read

온라인 연결

등록된 사설서버가 있다면 앱 시작과 동시에 연결을 시도합니다. 등록된 사설서버가 없고, 사설서버를 만들 의향이 없다면 상태등을 눌러 개발자가 사용하는 개발 테스트 서버를 이용할 수 있습니다. 이 서버는 글자 그대로 개발용 서버로 공식적으로 제공되는 서비스 서버가 아니므로 개발 내용 및 서버 상황 등에 따라 언제든 서버 초기화/오프라인 등의 조치가 이루어질 수 있습니다.

서버에 연결할 때 사용하는 계정 정보는 이메일/비밀번호로, 실제 사용하는 이메일인지는 검토하지 않습니다. 그러므로 aa@aa.aa 와 같이 이메일 형식만 맞춰도 가입되어 사실상 익명으로 서비스 이용이 가능합니다. 비밀번호는 최소 8자리 이상으로 설정해야 합니다.

여러개의 사설서버에 연결할 때 하나의 계정정보(이메일/비밀번호)를 이용하게 됩니다. 사설서버를 삭제하면 회원탈퇴를 진행하며 계정으로 활동한 모든 정보를 서버에서 삭제합니다.

채널 채팅 생성 및 참여

채널을 생성하면 같은 서버를 사용하는 사람들과 대화할 수 있습니다. 생성된 채널 설정에 있는 QR코드를 인식하거나 QR코드 아래 있는 채널 아이디를 입력하여 채널에 참여할 수 있습니다.

채널 생성 버튼은 온라인 표시등 왼쪽에 있습니다

채널을 삭제하면 해당 채널과 관련된 모든 정보를 서버에서 삭제합니다.

서버에 할 일 저장하기

서버가 추가된 상태에서 할 일을 추가하면 저장소를 지정할 수 있습니다.

이렇게 하면 다른 기기에서 로그인했을 때 서버로부터 할 일 목록을 불러올 수 있습니다.

할 일을 삭제하거나 완료하면 해당 할 일과 관련된 모든 정보를 서버에서 삭제합니다.

음성 대화 서버 연결

설정 > WebRTC 서버 메뉴에서 서버를 등록하면 1:1 채널에서 사용자간 음성대화를 할 수 있습니다.
