Project: Cone 앱 사용 방법 (3)

Posted Feb 16, 2024
By 최성수 3 min read

사설 서버 설치하기

설정 > 개발 보조 도구 > Nakama 서버 파일 메뉴를 선택하면 다운로드 폴더에 서버 설정 파일(nakama.zip)을 다운로드 받습니다. 설정 파일을 동작시키기 위해선 docker를 설치해야 합니다.

리눅스 패키지 매니저로 설치한다면 docker-compose 까지 설치해주셔야 합니다.

nakama 폴더 안에서 터미널을 실행한 후 다음 명령어를 입력하여 서버를 시작할 수 있습니다.

1

docker-compose up -d

컴퓨터의 ip주소를 확인하여 서버 주소에 입력하면 사설 서버에 접근할 수 있습니다.

이렇게하면 내부망에서 정상적으로 동작하게 됩니다.

외부망에서 접근할 수 있도록 구성하려면 https 설정이 필요한데, noip에서 무료 ddns를 발급받은 후 certbot으로 보안처리하는 것을 추천드립니다.

관리자 권한 설정

나카마 서버에 회원가입이 완료되었다면 사용자 계정 정보를 수정하여 관리자를 직접 지정할 수 있습니다. 관리자로 지정된 사람은 설정 메뉴에 서버 정보를 간략하게 읽을 수 있는 관리자 메뉴가 생성됩니다.

나카마 콘솔은 7351 포트로 진입할 수 있습니다.

내장 뷰어에서 열리는 Godot engine 3 패키지 파일 구성

앱 내장 뷰어에서 *.pck 파일을 열람하고 싶다면 ContentViewer.tscn 이라는 씬을 프로젝트 루트 에 생성한 후 아래 스크립트를 추가하고, 뷰어에서 바로 열리기 원하는 씬을 res://ShareTarget.tscn 대신 입력해주세요.

1
2
3
4
5
6

# ContentViewer.gd
extends Node

func _ready():
	var inst = load('res://ShareTarget.tscn')
	add_child(inst.instance())

인앱 뷰어는 *.pck 파일을 HTML5 에서 재생하므로 엔진에서 HTML5 출력시 발생하는 모든 제한사항이 적용됩니다. 또한 프로젝트 설정(project.godot 파일에 내장되는 내용들)에 해당하는 내용들(키맵, 렌더설정, 싱글톤 등)이 동작하지 않습니다.
